<h2>🛠️ Installation Steps:</h2>

<p>1. Install Bun JS Runtime</p>

```
powershell -c "irm bun.sh/install.ps1 | iex" (Windows)
```

<p>2. Clone the repository wherever you like</p>

<p>3. Install the packages</p>

```
bun install
```

<p>4. Run the NextJS server</p>

```
bun run dev
```
